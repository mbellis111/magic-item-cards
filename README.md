# Magic Item Cards

This repo creates a react app to create, manage, and print magic item cards for tabletop games.

## Requirements

```
npm --version
11.6.2

node --version
v24.12.0

pnpm --version
10.27.0
```

## Setup

1. Run `pnpm install`
2. Run locally with `pnpm dev`
3. View at http://localhost:5173/

Review [package.json](/.package.json) for additional commands.

## React Compiler

The React Compiler is enabled on this template. See [this documentation](https://react.dev/learn/react-compiler) for more information.

Note: This will impact Vite dev & build performances.
